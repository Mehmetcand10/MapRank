# MapRank

MapRank is a B2B SaaS platform for Google Maps SEO analysis and ranking intelligence.

## Technology Stack

- **Backend:** Python (FastAPI), PostgreSQL, Celery, Redis.
- **Frontend:** Next.js, TypeScript, TailwindCSS.
- **Infrastructure:** Docker, Railway/Render, Vercel.

See `backend/README.md` and `frontend/README.md` for specific instructions.
